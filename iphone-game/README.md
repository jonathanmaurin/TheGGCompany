# Stack Attack — iPhone Game

Phaser 3 game wrapped in Capacitor for iOS distribution.

## Local Dev

```bash
cd iphone-game
npm install
npm run dev        # Opens at http://localhost:3000 (390px wide)
```

## Build for iOS

### Prerequisites
- Xcode installed (App Store or developer.apple.com)
- CocoaPods: `sudo gem install cocoapods`
- Capacitor CLI: included as devDependency

### Steps

```bash
# 1. Build web assets
npm run build

# 2. Sync to iOS project
npx cap sync

# 3. Open in Xcode
npx cap open ios
```

### First-time iOS setup (if ios/ doesn't exist)

```bash
npx cap add ios
```

Note: Requires Xcode.app (not just Command Line Tools). CocoaPods must be installed for `pod install` to run during sync.

## Game Config

- **App ID:** com.theggcompany.stackattack
- **Canvas size:** 390×844 (iPhone 14 Pro / iPhone 15)
- **Engine:** Phaser 3

## Deployment

1. Run `npm run build && npx cap sync`
2. Open in Xcode: `npx cap open ios`
3. Set signing team in Xcode → Select target → Signing & Capabilities
4. Build & Run on device or Archive for App Store
